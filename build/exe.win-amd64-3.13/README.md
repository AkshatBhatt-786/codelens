# CodeLens - Java Learning Platform

A terminal-based Java code learning platform with GitHub synchronization and community features.

## Features

- Browse and study Java code examples
- GitHub synchronization for personal repositories
- Community repository for shared learning
- Beautiful terminal UI with Rich
- Secure authentication system
- Real-time file monitoring

## Installation

### Option 1: From Source
```bash
git clone <repository-url>
cd codelens
pip install -r requirements.txt
python src/main.py
